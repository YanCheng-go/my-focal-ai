Personal news intelligence system that aggregates content from multiple sources, then scores relevance using LLM against user-defined principles.

How it works

The pipeline fetches content from Twitter/X, YouTube, arXiv, RSS feeds (via RSSHub), GitHub trending repos, and more. Each item is scored 0-100 against your priorities defined in config/principles.yml. Two modes: local (SQLite + Ollama + FastAPI) and cloud (GitHub Actions + Vercel static site).

Tech stack

  • BackendPython, FastAPI, APScheduler
  • DatabaseSQLite (WAL mode)
  • ScoringOllama (local) / Claude API (cloud)
  • IngestionRSSHub, aiohttp, BeautifulSoup
  • FrontendJinja2 templates (local), static HTML + JS (Vercel)
  • CI/CDGitHub Actions, Vercel
  • Dev toolsNix + direnv, uv, ruff
