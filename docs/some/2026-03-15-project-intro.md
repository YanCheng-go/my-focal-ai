**Date:** 2026-03-15

## Post

I built an open-source tool to cut through AI news noise.

MyFocalAI aggregates 50+ sources — Twitter, YouTube, RSS, arXiv, GitHub trending — and uses an LLM to score each item against your own relevance principles. No algorithm decides what matters. You do.

- Runs fully local with Ollama (free) or deployed to Vercel with Claude API
- You define "relevant" in a simple YAML file — the LLM scores, ranks, and explains why
- Built with Python, FastAPI, SQLite, Tailwind CSS

GitHub: https://github.com/YanCheng-go/my-focal-ai
